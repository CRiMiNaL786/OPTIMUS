# Deploy:
## Environment Variables
---
- `API_HASH` Your API Hash from my.telegram.org
- `API_ID` Your API ID from my.telegram.org
- `SESSION_STRING` Your PYROGRAM String Session
---

## Using Heroku
---
- Click Deploy Icon
- Give App Name And Fill The ENV VARS.
- Click Deploy
